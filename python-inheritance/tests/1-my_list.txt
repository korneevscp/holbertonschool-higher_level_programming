======================

Using ``MyList``
-------------------


First import ``print_sorted`` from the ``1-my_list`` file:

    >>> ps = __import__('1-my_list').MyList.print_sorted

Tests:

    Test - check that sorting works
    >>> ps([1, 4, 2, 3, 5])
    [1, 2, 3, 4, 5]
    [1, 4, 2, 3, 5]

    Test - list contains non-integer values (string)
    >>> ps(["haha", 2, 1])

    Test - list contains float values
    >>> ps([4.5, 2, 1])

    [4.5, 2, 1]

    Test - list is empty
